批量视频组合拼接工具 发布说明

一、适用对象

本发布包适用于普通 Windows 用户。

用户不需要单独安装 Python。
只需要解压压缩包后，双击程序即可启动。


二、压缩包内容

压缩包中包含：

1. BatchVideoConcatTool.exe
2. _internal 运行环境目录
3. 用户使用说明.txt

请务必保留完整目录结构，不要只拿出 exe 单独使用。


三、用户如何启动

1. 解压整个压缩包
2. 打开解压后的文件夹
3. 双击：

   BatchVideoConcatTool.exe

即可启动程序。


四、发给别人时怎么发

推荐直接发送下面这个压缩包：

- BatchVideoConcatTool-win64.zip

不要只发送 exe。
因为程序运行需要同目录下的 `_internal` 文件夹。


五、如果用户打不开

请先让用户检查：

1. 是否完整解压了整个压缩包
2. 是否保留了 `_internal` 文件夹
3. 是否在 Windows 上运行
4. 是否被杀毒软件或系统安全提示拦截


六、当前版本特性

当前版本支持：

- 导入多个完整视频素材
- 设置 K 和导出任务数量
- 预览 N / G / L1 / L2 / L3 / T
- 选择视频输出目录
- 选择导出比例
- 导出 DaVinci Resolve Lua 脚本
- 在 Resolve 中手动执行脚本完成批量渲染


七、当前版本限制

当前版本仍然需要用户手动在 DaVinci Resolve 中执行 Lua 脚本。

也就是说：

1. 先在本软件里导出脚本
2. 再到 DaVinci Resolve 的 Lua Console 中运行脚本


八、备注

如果后续需要更正式的安装方式，可以继续制作：

- 安装包（例如 Inno Setup）
- 桌面快捷方式
- 开始菜单入口

当前这个 zip 发布包已经适合直接发给别人测试和使用。
